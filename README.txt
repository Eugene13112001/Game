Для запуска программы необходимо заменить строку подключения к БД в appsettings.json .
Реализованы следующие методы:
1. AddGame 
Добавление новой игры . Вводятся название игры , название создателя и массив с жанрами игры.
Возвращает true в случае успеха операции, в случае возникновения ошибки - false.
2. AddView
Добавление новго жара игры.  Вводится только название нового жанра.
Возвращает true в случае успеха операции, в случае возникновения ошибки - false.
3. GetGameAll
Получение списка всех игр.
Возвращает  список всех игр  в формате JSON.
4. GetGameBy
Получение списка  игр, которые соотвествуют определенному жанру.
Вводится название интересующего нас жанра и возвращается список всех игр, которые ему соответствуют, в формате JSON.
5. GetViews
Возвращает  список всех жанров в формате JSON.
6. ChangeView
Изменение названия жанра. Вводится id жанра и новое назваание.
Возвращает true в случае успеха операции, в случае возникновения ошибки - false.
7. ChangeGame
Изменение информации о игре. Вводится id игры ,  новое название, новое название создателя,  а также массив с новыми жанрами игры.
Возвращает true в случае успеха операции, в случае возникновения ошибки - false.
8. DeleteView
Удаление жанра. Вводится id жанра , после чего он удаляется.
Возвращает true в случае успеха операции, в случае возникновения ошибки - false.
9. DeleteGame
Удаление игры. Вводится id игры, после чего она удаляется.
Возвращает true в случае успеха операции, в случае возникновения ошибки - false.
